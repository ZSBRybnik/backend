---
title: Tabela - produkty
---

### Kolumny

- `id` - reprezentuje unikalny ID użytkownika. Jest generowane automatycznie. Nie może być nullem. Wartość jest liczbą typu całkowitego.
- `name` - reprezentuje unikalną nazwę produktu. Nie może być nullem. Wartość jest typu tekstowego oraz nie może być dłuższa niż 100 znaków.
- `price` - reprezentuje cenę produktu w złotówkach. Nie może być nullem. Wartość jest typu dziesiętnego
- `quantity` - reprezentuje ilość produktu na stanie. Nie może być nullem. Wartość jest liczbą typu całkowitego
- `description` - reprezentuje opis produktu. Nie może być nullem. Wartość jest typu tekstowego oraz jego długość nie jest ograniczona.
