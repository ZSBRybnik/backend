## Kolumny

- `name` - reprezentuje unikalną nazwę strony. Nie może być nullem. Wartość jest typu tekstowego o maksymalnej długości 255 znaków.
- `category` - reprezentuje kategorię do której należy strona. Wartość jest typu tekstowego o maksymalnej długości 255 znaków.
- `title` - reprezentuje tytuł strony. Wartość jest typu tekstowego o maksymalnej długości 255 znaków.
