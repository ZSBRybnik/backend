### Kolumny

- `id` - reprezentuje unikalny ID przedmiotu. Jest generowane automatycznie. Nie może być nullem. Wartość jest liczbą typu całkowitego.
- `name` - reprezentuje nazwę przedmiotu. Wartość jest typu tekstowego o maksymalnej długości 255 znaków.
