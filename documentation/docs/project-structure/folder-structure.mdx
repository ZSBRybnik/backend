## Katalog główny projektu

- `.github` - W tym folderze znajduje się konfiguracja zewnętrznego repozytorium na GitHubie.
- `.husky` - Folder ten jest odpowiedzialny za konfigurację biblioteki husky, która pozwala na zautomatyzowanie zadań podczas pracy z gitem.
- `.storybook` - Folder ten jest odpowiedzialny za konfigurację biblioteki Storybook do testowania oraz dokumentowania wszelakich komponentów używanych w emailach.
